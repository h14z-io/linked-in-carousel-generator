# Template Images

This folder contains the visual template images for the 7 narrative styles.

## Required Templates (14 total)

Each narrative style needs 2 versions (dark + light theme):

### 1. PASP Framework (Problem-Agitation-Solution-Proof)

- `pasp-dark.png`
- `pasp-light.png`
- **Style**: Bold borders, urgent feel, strong lines, high contrast

### 2. BAB Framework (Before-After-Bridge)

- `bab-dark.png`
- `bab-light.png`
- **Style**: Split screen feel, before/after visual hint, transformation theme

### 3. Step-by-Step Guide

- `step-by-step-dark.png`
- `step-by-step-light.png`
- **Style**: Numbered zones, progression indicators, clean educational layout

### 4. Data Storytelling

- `data-storytelling-dark.png`
- `data-storytelling-light.png`
- **Style**: Grid pattern, chart-like decorative elements, space for large numbers

### 5. Listicle/Tips

- `listicle-dark.png`
- `listicle-light.png`
- **Style**: Playful, bullet-friendly zones, space for emojis, energetic

### 6. Behind-the-Scenes

- `behind-the-scenes-dark.png`
- `behind-the-scenes-light.png`
- **Style**: Organic, handwritten feel, subtle texture, authentic vibe

### 7. Case Study

- `case-study-dark.png`
- `case-study-light.png`

## Specifications

- **Size**: 1080x1080px (1:1 square ratio)
- **Format**: PNG (with transparency support if needed)
- **Theme Colors**:
  - Dark: Background `#0B0B0E`, Primary accent `#BB2649`
  - Light: Background `#FFFFFF`, Primary accent `#BB2649`
- **Space Guidelines**:
  - Leave clear areas for text content
  - Margins: min 60px from edges
  - Suggested zones:
    - Header area (top): ~150px height
    - Title area: ~180px height
    - Content area (middle): ~500px height
    - Footer area (bottom): ~60px height

## Design Notes

- Keep decorative elements subtle (borders, shapes, gradients, textures)
- Ensure high readability for text overlay
- Maintain visual consistency within theme (dark/light)
- Each narrative style should have distinct visual identity
- Professional LinkedIn aesthetic (not playful social media style)

## Placeholder Templates

Until final designs are ready, you can use solid color backgrounds:

- Dark: `#0B0B0E`
- Light: `#FFFFFF`

The AI will still generate content correctly with placeholder templates.
